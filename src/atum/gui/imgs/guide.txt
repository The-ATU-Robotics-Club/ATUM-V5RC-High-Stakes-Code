1. Go here: https://lvgl.io/tools/imageconverter
2. Select v8
3. Upload the image
4. Give an appropriate name
5. Select "True color with alpha"
6. Select "C array"
7. Replace 
        #if defined(LV_LVGL_H_INCLUDE_SIMPLE)
            #include "lvgl.h"
        #else
            #include "lvgl/lvgl.h"
        #endif
   with 
        #include "liblvgl/lvgl.h"
8. Add to the appropriate folder
