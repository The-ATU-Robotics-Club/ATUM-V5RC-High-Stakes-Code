Detailed Description

template<typename Unit>
class atum::Acceptable< Unit >

This class template produces encapsulates several conditions for the results of an action being considered acceptable, including a timeout (past which the action is considered acceptable), a max error (below which, the action is considered acceptable), a max derivative of error (below which, assuming error is fine, the action is considered acceptable) in order to account for momentum, and a minimum time to meet those conditions for.
