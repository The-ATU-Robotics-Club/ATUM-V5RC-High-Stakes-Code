Detailed Description

template<typename Unit>
class atum::MotionProfile< Unit >

This class supports the efficient creation of S and trapezoidal motion profiles, alongside helpful logging and graphing functionalities.

Can be polled by position to allow for robust following. This works by comparing the closest point in time and position along the profile and using the point with the larger velocity.
