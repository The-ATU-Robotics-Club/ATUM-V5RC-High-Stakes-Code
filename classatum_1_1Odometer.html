Detailed Description

This class represents an encoder (here an AMT102-V encoder) with a wheel, used to measure traveled distance. This is an essential component in odometry.

Distance from center can be easily determined via a CAD model. Wheel circumference can be measured by getting the raw readings from the encoder and moving the robot a known distance; then, simply work backwards to calculate circumference.
